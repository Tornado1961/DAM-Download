# DAM Download

Repository pubblico minimale per distribuire build di prova DAM Field.

## Contenuto

- `index.html`: pagina download.
- `assets/site.css`: stile della pagina.
- `downloads/DAM_Field_Ver_1.01.048-release.apk`: APK DAM Field release firmato per collaudo.
- `downloads/DAM_Field_Ver_1.01.048-release.apk.sha256`: checksum SHA-256.

Versione corrente: `1.01.048`

SHA-256: `346127579A9FFCE96DEE23FEE5FA5956F7CFE9B8ACBE259ACF1A5414CA4D0D2C`

## Sicurezza

Questo repository non contiene codice sorgente DAM, archivi cliente, dati operativi, configurazioni private o token.

## GitHub Pages

Per pubblicare:

1. Aprire `Settings`.
2. Aprire `Pages`.
3. Selezionare `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/root`.
6. Salvare.

URL atteso: `https://tornado1961.github.io/DAM-Download/`.
