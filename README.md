# DAM Download

Repository pubblico minimale per distribuire build di prova DAM Field.

## Contenuto

- `index.html`: pagina download.
- `assets/site.css`: stile della pagina.
- `downloads/DAM_Field_Ver_1.01.053-release.apk`: APK DAM Field release firmato per collaudo.
- `downloads/DAM_Field_Ver_1.01.053-release.apk.sha256`: checksum SHA-256.

Versione corrente: `1.01.053`

SHA-256: `E32E6DB70A395FDD77369A0762EA6FAD3FFF53D5795AC275ACB8E266A56D6227`

## Sicurezza

Questo repository non contiene codice sorgente DAM, archivi cliente, dati operativi, configurazioni private o token.

## GitHub Pages

Per pubblicare:

1. Aprire `Settings`.
2. Aprire `Pages`.
3. Selezionare `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/root`.
6. Salvare.

URL atteso: `https://tornado1961.github.io/DAM-Download/`.
